# 🔥 Welding Services Website (React + Vite)

A **modern, cinematic and professional landing page** built for a **welding and metal fabrication workshop**.
This website showcases welding services, tools, gallery work and allows customers to contact the workshop easily.

The project is designed with a **dark industrial theme** and **cinematic UI** using modern frontend technologies.

---

# 📌 Project Overview

This website is created for a **local welding workshop / fabrication business** where customers can:

* Explore available **welding services**
* See **fabrication work gallery**
* Learn about the workshop
* View **tools and welding equipment**
* Contact the workshop for **custom work**

The design follows a **dark industrial theme** with orange highlights which represents **heat, sparks and welding fire**.

---

# 🎨 Website Features

### ⭐ Cinematic UI Design

* Dark industrial color theme
* Orange accent highlights
* Professional typography
* Smooth hover animations

### ⭐ Single Page Landing Layout

All sections exist in a **single landing page**:

* Hero Section
* About Section
* Tools Section
* Services Section
* Work Gallery
* Contact Form
* Footer

Navigation scrolls smoothly between sections.

---

### ⭐ Smooth Scroll Navigation

Navbar links scroll directly to sections using **HTML IDs and smooth scrolling**.

Example:

```
Home → About → Services → Gallery → Contact
```

---

### ⭐ Welding Services Section

Shows the main services offered:

* Arc Welding
* Metal Fabrication
* Gate & Grill Work
* Steel Structure Work
* Machine Repair
* Custom Welding

Includes a **cinematic welding video** to visually demonstrate the work process.

---

### ⭐ Tools Section

Displays tools used in welding such as:

* Welding machine
* Cutting tools
* Steel fabrication tools
* Industrial equipment

---

### ⭐ Work Gallery (Pinterest Style)

A modern **masonry gallery layout** that shows:

* Iron gates
* Steel railings
* Window grills
* Metal sheds
* Iron doors
* Balcony grills

Images appear **black & white initially** and reveal color on hover.

---

### ⭐ Contact Form

Customers can send work requests by filling:

* Name
* Phone Number
* Work Details
* Message

This helps customers contact the welding shop easily.

---

### ⭐ Cinematic Footer

The footer includes:

* Workshop description
* Services list
* Contact information
* Quick navigation links

# 🚀 Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone <repository-url>
```

---

### 2️⃣ Install dependencies

```
npm install
```

---

### 3️⃣ Start development server

```
npm run dev
```

The project will run at:

```
http://localhost:5173
```
